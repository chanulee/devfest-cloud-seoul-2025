# ADK GO
## Setting up

- download go for your device 
- go extension download in your IDE
- ``go get google.golang.org/adk``
- Set up API key from Google AI Studio
- Set up billing in Google Cloud Console (Connect the key with billing info/credit)

## Concepts
- Model
- Agent
- Tool: This includes actual tool calling and the output structure schema
- Launcher: utility that provides the necessary runtime environment and services to execute and test your AI agent
